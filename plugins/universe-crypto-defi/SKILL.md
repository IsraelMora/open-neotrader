## Características del mercado DeFi

```
1. Trading 24/7/365 — sin cierres de mercado ni gaps de weekend
2. Alta volatilidad — movimientos 3-10x más amplios que equities
3. Correlación con BTC/ETH — en pánicos el sector cae en bloque
4. Smart money flows — eventos: governance, unlock de tokens, hacks
5. On-chain transparency — datos en blockchain (TVL, fees, usuarios)
```

## Factores fundamentales DeFi

- **TVL (Total Value Locked)**: liquidez depositada en el protocolo
  - Métrica clave de adopción; ratios precio/TVL similares al P/S
- **Protocol Revenue**: fees cobradas; mide rentabilidad real
- **Token Utility**: ¿el token captura valor del protocolo?
  - Governance puro (bajo valor) vs. fee sharing (alto valor)
- **Emission Schedule**: tokens que se crean inflan el supply y presionan precio

## Correlaciones típicas (bear vs bull)

```
Bull market:
- DeFi tokens x3-x10 vs BTC cuando BTC rompe máximos
- Rotación: BTC → ETH → altcoins grandes → DeFi → memes

Bear market / pánico:
- Correlación con BTC: 0.85-0.95 (todos caen juntos)
- Protocolo con hack: -40% en minutos

Neutral:
- Correlación media: 0.5-0.7 con BTC
```

## Risks específicos DeFi

```
1. Smart contract risk: bugs → hack → precio -50% en horas
   Mitigación: solo tokens con auditorías múltiples (Certik, Trail of Bits)

2. Liquidity risk: DEX con pool pequeño → gran impacto precio
   Mitigación: usar solo pares con >$10M liquidez on-chain

3. Regulatory risk: SEC puede clasificar token como security
   Mitigación: seguir categoría de CFTC vs SEC en noticias

4. Bridge risk: tokens cross-chain tienen riesgo adicional
   Mitigación: preferir tokens nativos en su L1/L2 original
```
