# Alpaca Provider

Proveedor de datos y broker para renta variable (equities) vía la API de Alpaca Markets.

## Credenciales requeridas

- `ALPACA_API_KEY_ID` — Key ID de tu cuenta Alpaca
- `ALPACA_API_SECRET_KEY` — Secret de tu cuenta Alpaca

## Tools disponibles

Consulta `tools.json` en este directorio para la lista completa de funciones llamables.

## Modos de operación

- **paper**: Operaciones simuladas (recomendado para pruebas)
- **live**: Operaciones reales — requiere cuenta financiada
